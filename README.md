# CloudCode Editor
Lightweight Desktop Code Editor built with Python & CustomTkinter, compiled by Nuitka.

## Features
- Modern dark‑themed UI
- Multi‑language support: Python, HTML, CSS, JavaScript, Java, C, C++, C#, Kotlin, Plain Text
- Built‑in local HTTP server for HTML live preview
- Complete file manager: create, save, rename, delete, refresh files/folders
- Custom workspace directory
- One‑click EXE packaging
- Ultra‑small size & fast startup (Nuitka compiled, way smaller than PyInstaller)
- Pure desktop‑oriented, stable performance
