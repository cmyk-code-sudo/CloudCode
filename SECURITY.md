# Security Policy
## Supported Versions
Only the official original release is supported.
All modified third‑party versions are not safe and not supported.

## Reporting a Vulnerability
If you find security vulnerabilities, contact the author via email: wwwbibiqqcom@qq.com

## Warning
Any unofficial modified version may contain viruses, trojans, or spyware.
The original author is not responsible for any losses caused by unofficial versions.
