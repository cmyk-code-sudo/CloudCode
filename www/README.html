CloudCode Editor

Lightweight Cross‑Platform Desktop Code Editor built with Python & CustomTkinter, compiled by Nuitka with ultra‑small size.

✨ Core Features

  • Modern dark‑themed UI interface
  • Multi‑language support: Python, HTML, CSS, JavaScript, Java, C, C++, C#, Kotlin, Plain Text
  • Built‑in local HTTP server for real‑time HTML live preview
  • Complete file manager: create, open, save, rename, delete, refresh files/folders
  • Custom workspace directory support
  • One‑click EXE packaging function
  • Ultra‑small size & fast startup speed (Nuitka compiled)
  • Stable performance, low memory usage

🧰 Requirements

  • Python ≥ 3.10
  • customtkinter
  • nuitka (compilation tool)

📦 Build Command

py -m nuitka --onefile --standalone --windows-disable-console --output-filename=CloudCodeEditor src/main.py
